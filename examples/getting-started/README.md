# Getting Started

## Purpose

This directory contains introductory examples for developers new to prompt engineering. These scripts demonstrate the simplest patterns for getting started with OpenAI's API.

## Examples

### script.py

**Lesson:** Create your first prompt with clear instructions and audience targeting.

**What you'll learn:**
- How to write a basic prompt with specific requirements
- How to target output for a specific audience (e.g., children)
- Basic prompt structure and formatting

**Run from root:**
```bash
python examples/getting-started/script.py
```

**Key takeaway:** Always specify your desired output format and target audience to get more relevant, tailored responses. Clear instructions lead to better results.
